# Calculadora Educativa de Matrices

## Descripción

Este programa es una herramienta educativa diseñada para el curso de Fundamentos de Programación. Implementa múltiples operaciones con matrices, cada una explicada desde su base teórica hasta su implementación práctica.

## Conceptos Cubiertos

1. **Matrices**: Definición, tipos y propiedades
2. **Operaciones básicas**: Suma, resta, multiplicación
3. **Determinantes**: Métodos de cálculo y significado geométrico
4. **Matriz inversa**: Cálculo y aplicaciones
5. **Sistemas de ecuaciones**: Métodos de resolución
6. **Valores propios**: Teoría y aplicaciones
7. **Descomposiciones**: LU, QR
8. **Aplicaciones prácticas**: Regresión lineal, transformaciones

## Cómo usar

1. Compile todos los archivos: `g++ -o calculadora *.cpp -std=c++11 -lm`
2. Ejecute: `./calculadora`
3. Siga las instrucciones del menú
